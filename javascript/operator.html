01. 산술 연산자

산술 연산자의 종류는 다음과 같습니다.

산술 연산자 종류 속성 설명
+ 더하기 연산을 할 때 사용
문자와 문자, 문자와 변수 등을 연결할 때 사용
- 빼기 연산을 할 때 사용
* 곱하기 연산을 할 때 사용
/ 몫을 구할 때 사용
% 나머지 구할 때 사용
++ 1씩 증가 시킬 때 사용
-- 1씩 감소 시킬 때 사용

02. 대입 연산자

대입 연산자의 종류는 다음과 같습니다. 대입 연산자의 '='는 '같다'를 의미하지 않으며 변수에 대입(저장)함을 의미합니다.

대입 연산자 종류 속성 설명
= num = 1은 우변의 1값을 좌변의 num 변수에 대입(저장)
+= num += 2는 num = num + 2와 같음
-= num -= 2는 num = num - 2와 같음
*= num *= 2는 num = num * 2와 같음
/= num /= 2는 num = num / 2와 같음
%= num %= 2는 num = num % 2와 같음

03. 비교 연산자

비교 연산자는 값을 비교하여 결과값을 Boolean(논리)값인 true나 false로 반환합니다.

비교 연산자 종류 속성 설명
> a > b가 참이면 true, 거짓이면 false
< a < b가 참이면 true, 거짓이면 false
>= a >= b가 참이면 true, 거짓이면 false
<= a <= b가 참이면 true, 거짓이면 false
== a == b, a와 b가 같으면 true, 같지 않으면 false.
== 연산자의 경우 데이터 타입은 비교하지 않음.
--> ex) 5 == '5'는 true가 나옴.
!= a != b, a와 b가 같지 않으면 true, 같으면 false.
=== a === b, a와 b가 같고 데이터 타입도 같으면 true, 그렇지 않으면 false. --> ex) 5 == '5'는 false가 나옴.
!== a !== b, a와 b가 같지 않거나 데이터 타입이 같지 않으면 true, 같으면 false.

이상, 초과, 이하, 미만 표현은 다음과 같이 합니다.

이상 num >= 10
초과 num > 10
이하 num <= 10
미만 num < 10

04. 논리 연산자

논리 연산자의 종류는 다음과 같습니다.

논리 연산자 종류 설명
&& (AND 연산자) a && b, a와 b 모두 참일 때 true가 됨.
--> true && true = true / true && false = false
--> false && true = false / false && false = false
|| (OR 연산자) a || b, a와 b 중 하나만 참이면 true가 됨.
--> true || true = true / true || false = true
--> false || true = true / false || false = false
! (NOT 연산자) !a, a가 true면 false, false면 true로 바꿔줌.

4-1. 논리 연산자를 이용한 부등식 표현

'x의 값이 20보다 크고 30보다 작다.'를 일반적 부등식 표현으로 '20 < x < 30'이지만 자바스크립트에서는 'AND 연산자'를 사용하여 표현합니다.
