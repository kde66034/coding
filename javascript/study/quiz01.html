퀴즈 만들기

이번 시간에는 그 동안 배웠던 JavaScript를 이용하여 퀴즈를 만들어보도록 하겠습니다.
변수(let)와 상수(const)에 쿼리 셀렉터 값을 저장시키고, 그 동안 배운 스크립트를 통해 문제 정보를 HTML 태그에 출력시키고, 정답을 숨겼다가 버튼을 눌러 보이게 하는 등의 다양한 효과들을 재현시켜봅시다!

01. 선택자

상수 const를 필요한 만큼 선언시킨 다음 적절한 이름을 붙여주고, 쿼리 셀렉터를 통해 연결 시켜주고자 하는 CSS 속성명을 아래의 예시와 같이 입력하여 연결 작업을 하도록 합시다. 

//const 적절한 이름 = document.querySelector("연결시켜주고자 하는 CSS 속성명");

const quizType = document.querySelector(".quiz__type");                             // 퀴즈 종류
const quizNumber = document.querySelector(".quiz__question .number");               // 퀴즈 번호
const quizAsk = document.querySelector(".quiz__question .ask");                     // 퀴즈 질문
const quizConfirm = document.querySelector(".quiz__answer .confirm");               // 정답 확인 버튼
const quizResult = document.querySelector(".quiz__answer .result");                 // 정답 결과
const quizInput = document.querySelector(".quiz__answer .input");                   // 사용자 정답
const quizView = document.querySelector(".quiz__view");

02. 문제정보 출력하기

// 문제 정보
const answerType = "웹디자인기능사";    // 문제 타입
const answerNum = "1";      // 문제 번호
const answerAsk = "기업의 새로운 이념 구축에 필요한 이미지와 커뮤니케이션 시스템을 의도적, 계획적으로 만들어내는 기업이미지통합 전략은?";      // 질문
const answerResult = "CIP";        // 정답

// 문제 출력 (innerText 대신 textContent를 사용해도 됨 / 익스플로어 호환되는 건 textContent)
quizType.innerText = answerType;
quizNumber.innerText = "0" + answerNum + ". ";
quizAsk.innerText = answerAsk;
quizResult.innerText = "정답은 [ " + answerResult + " ] 입니다.";
