indexOf()를 이용하여 검색하기

HTML 태그 검색하기

  • a : 다른 페이지로의 이동을 설정함
  • abbr : 줄임말이나 머리글자를 표현 할 때 사용
  • acronym : 줄임말이나 머리글자를 표현 할 때 사용
  • address : 가까운 HTML 요소의 사람, 단체, 조직 등에 대한 연락처 정보를 나타냅니다.
  • applet : 사용되지 않음
  • area : 이미지의 핫스팟 영역을 정의하고 하이퍼링크 (en-US)를 추가할 수 있습니다. map 요소 안에서만 사용할 수 있습니다.
  • article : 문서, 페이지, 애플리케이션, 또는 사이트 안에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획을 나타냅니다.
  • aside : 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다.
  • audio : 문서에 소리 콘텐츠를 포함할 때 사용합니다.
  • b : 독자의 주의를 요소의 콘텐츠로 끌기 위한 용도로 사용합니다.
  • base : 문서 안의 모든 상대 URL이 사용할 기준 URL을 지정합니다.
  • bdi : bdi 태그는
  • bdo : 현재 텍스트의 쓰기 방향을 덮어쓰고 다른 방향으로 렌더링 할 때 사용합니다.
  • bgsound : 배경 사운드 설정. 현재 사용되지 않음.
  • big : big. 현재 사용되지 않음.
