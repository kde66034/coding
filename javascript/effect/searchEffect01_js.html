indexOf()를 이용하여 검색하기

JavaScript 메서드 검색하기

  • indexOf : 문자열에서 특정 문자의 위치를 찾고, 숫자를 반환합니다.
  • slice : 문자열에서 원하는 값을 추출해 문자열을 반환하는 메서드.
  • substring : 문자열에서 원하는 값을 추출해 문자열을 반환하는 메서드. substring() 시작값이 끝나는 값보다 클 경우 두 값을 바꿔서 처리(에러 방지)
  • substr : 문자열에서 원하는 값을 추출해 문자열을 반환하는 메서드.
  • split : 문자열에서 원하는 값을 배열로 반환합니다.
  • trim : 문자열의 앞·뒤 공백을 제거합니다.
