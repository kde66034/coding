CSS 속성 검색하기

  • accent-color : 다른 페이지로의 이동을 설정함
  • align-content : 콘텐츠 아이템의 상하관계 정렬 상태를 설정
  • align-items : 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정함
  • align-self : 개별적 콘텐츠 아이템의 정렬 상태를 설정함
  • all : 요소의 속성을 초기화 또는 상속을 설정함
  • animation : 애니메이션과 관련된 속성을 일괄적으로 설정함
  • animation-delay : 애니메이션 지연 시간을 설정함
  • animation-direction : 애니메이션 움직임 방향을 설정함
  • animation-delay : 애니메이션 움직임 시간을 설정함
  • animation-fill-mode : 애니메이션이 끝난 후의 상태 설정함
