사이트 만들기 : 카드 유형 02

이번 포스팅에서는 Figma에서 만든 카드 유형 섹션을 HTML과 CSS 코드로 옮기는 작업을 해보도록 하겠습니다.
최대한 웹 표준을 준수하기 위해 시맨틱 태그를 사용하고 이미지 출력 시 figure 태그로 감싸며, img 태그에서 alt(설명)칸에 부연 설명을 넣는 등의 작업을 잊지 말도록 합시다!

Figma

먼저 만들고자 하는 웹 사이트의 모양을 Figma라는 프로그램을 이용하여 아래와 같이 만들어주도록 하자.
이번에 만들 웹 사이트의 유형은 카드(8개) 유형 섹션이다.

CSS 소스

reset에서 margin값과 padding값을 0으로 초기화 시키는 작업부터 시작한다.
자세한 설명은 아래 예시 코드의 주석처리된 내용을 참조할 것. 

<style>
    /* fonts */
    @import url('https://webfontworld.github.io/gmarket/GmarketSans.css');

    .gmarket {
        font-family: 'GmarketSans';
        font-weight: 400;
    }

    /* reset */
    * {
        margin: 0;
        padding: 0;
    }
    h1, h2, h3, h4, h5, h6 {
        font-weight: normal;
    }
    a {
        text-decoration: none;
        color: #000;
    }
    img {
        width: 100%;    /* 이미지 튀어나온 거 집어넣기 */
    }

    /* common */
    .container {
        width: 1160px;
        padding: 0 20px;
        margin: 0 auto;
        min-width: 1160px;  /* 반응형 할 때 없애줄 것 */
    }
    .section {
        padding: 120px 0;
    }
    .section > h2 {
        font-size: 50px;
        line-height: 1;
        text-align: center;
        margin-bottom: 20px;
    }
    .section > p {
        font-size: 22px;
        font-weight: 300;
        text-align: center;
        line-height: 1.4;
        margin-bottom: 70px;
        color: #666;
    }

    /* cardType */
    .card__inner {
        display: flex;
        justify-content: space-between;     /* 간격 떨어뜨리기 */
        margin-bottom: 50px;
    }
    .card {
        width: 275px;
    }
    .card__header {
        padding-bottom: 20px;
    }
    .card__body {
        
    }
    .card__body .title {
        font-size: 28px;
        padding-bottom: 10px;
        position: relative;     /* 우측에 달린 버튼 고정할 위치 지정 */
    }
    .card__body .desc {
        font-size: 18px;
        font-weight: 300;
        line-height: 1.4;
        color: #666;
    }
    .card__body .btn {
        position: absolute;     /* 버튼 위치를 title쪽으로 고정시킴 */
        right: 0;
    }
</style>
